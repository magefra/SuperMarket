# SuperMarket
 API REST construido con .net core 3.0
 
 
 ## Frameworks and Libraries
 - NET.NET Core 3.0
 - Entity Framework Core
 - Entity Framework In-Memory Provider
 - AutoMapper
 
 ## Como empezar.
 
 Primero, iniciamos el proyecto.
 
 ```
dotnet restore
dotnet run
```
 
 Navegamos a https://localhost:5001/api/categories.
